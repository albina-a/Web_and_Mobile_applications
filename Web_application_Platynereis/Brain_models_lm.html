Interactive model of the Platynereis gene expression atlas

Drag the mouse to rotate the model and scroll to zoom

Platynereis gene expression atlas was generated basedon the in situ hybridization and light microscopy imaging using image registration. The model was generated using the 3D graphics software Blender. The cell-specific gene expression domains were approximated with the spheres and large expression domains were represented as smooth surfaces. Blender file can be downloaded from here.
